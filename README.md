# Unit Converter App

This is a simple **Unit Converter** web application made with **HTML, CSS and JavaScript**.

The app converts values between different units using one base unit for each category.

---

## Features

- Convert units easily
- Categories:
  - Mass (kg, g, lb)
  - Length (m, cm, ft, in)
  - Temperature (°C, °F, K)
  - Volume (L, mL, gal)
  - Time (h, min, s)
- Real-time conversion
- Simple and clean UI

---

## How It Works

1. User selects a category
2. User selects **from** and **to** units
3. User enters a value
4. Value is converted:
   - First to a **base unit**
   - Then to the target unit

Example:
Gram → Kilogram → Pound

## Rules

- All conversions use a base unit
- Invalid values are not converted
- Temperature below absolute zero is blocked
